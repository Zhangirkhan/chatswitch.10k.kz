# UI-чеклист приёмки ChatSwitch

**Назначение:** ручная проверка перед сдачей / демо.  
**Окружение:** Chrome, последняя сборка (`npm run build`), реальный WhatsApp + Reverb.  
**Легенда:** `[ ]` — проверить · `⚠` — известная заглушка (не кликать на демо) · `✓` — ок

## 13. Сценарии для демо AI (UI + поведение)

Прогнать в **одном** тестовом чате с включённым AI и воронкой:

| # | Действие клиента | Ожидание в UI | OK |
|---|------------------|---------------|-----|
| D1 | «Здравствуйте, что у вас есть?» | Ответ с каталогом, без «Спасибо за интерес» | [ ] |
| D2 | Уточнение цены | Короткий ответ, без повтора всего каталога | [ ] |
| D3 | «Запишите на 15:00, предупредите за 2 часа» | Подтвержение с датой `дд.мм в чч:мм`, этап воронки | [ ] |
| D4 | 3 сообщения подряд быстро | Один ответ, «печатает…» в WA | [ ] |
| D5 | Сложный вопрос без данных в БЗ | «Внимание» или вежливый запрос уточнения | [ ] |
