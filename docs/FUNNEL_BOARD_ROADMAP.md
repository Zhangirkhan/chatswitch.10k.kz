# Доска воронок — roadmap

## MVP (готово)

- Пункт «Воронки» в rail под календарём
- Kanban: выбор воронки, колонки этапов, карточки чатов
- Drag & drop смены этапа, клик → чат
- Фильтры: доступные / мои / мой отдел

## Пагинация (готово)

- Лимит 50 карточек на колонку при загрузке
- Кнопка «Показать ещё» + `GET /funnels/board/stage-cards`

## API

| Метод | URL |
|-------|-----|
| GET | `/funnels/board` |
| GET | `/funnels/board/data` |
| GET | `/funnels/board/stage-cards` |
| POST | `/funnels/board/bulk-move` |
| GET | `/funnels/board/card/{chat}` |
| PATCH | `/chats/{chat}/funnel` |
| GET | `/chats/{chat}/funnel/history` |

Query: `scope`, `assignee_id`, `department_id`, `whatsapp_session_id`, `search`, `funnel_id`.

Bulk-move body: `funnel_id`, `stage_id` (-1 = входящие), `chat_ids[]`, `force_locked?`.

Stage-cards query: `funnel_id`, `stage_id`, `offset`, фильтры как у `/data`.
